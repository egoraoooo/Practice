Задание №3. Написание функции-расширения
Реализован класс StringExtensions, который включает в себя 
функцию-расширение IsPalindrome для типа string, который проверяет,
является ли строка палиндромом.
Реализован класс StringExtensionsTests, в котором проверяется
правильность метода IsPalindrome.
Все тесты пройдены успешно.
