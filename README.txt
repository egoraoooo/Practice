Задание №5. Изучение класса с помошью рефлексии
Реализован класс ClassAnalyzer, который принимает 
тип класса в конструкторе.
Реализованы методы для изучения класса: GetPublicMethods(),
GetMethodParams(string methodname), GetAllFields(),
GetProperties(), HasAttribute<T>.
Реализован класс ClassAnalyzerTests, который позволяет
проверить работоспособность класса.
Все тесты пройдены успешно.
